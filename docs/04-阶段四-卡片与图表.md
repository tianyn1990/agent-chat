> **当前状态**：✅ 已完成实现。卡片/图表渲染引擎、Mock 数据触发关键词、单元测试（27 用例）均已就绪，`npm run test` 全部通过，`npm run build` 构建正常。

## 1. 目标

## 2. 交互式卡片设计

### 2.1 卡片 JSON 协议（简化版飞书卡片）

```typescript
// src/types/card.ts

interface InteractiveCard {
  type: 'interactive_card';
  cardId: string;          // 卡片唯一 ID（用于回传操作）
  header?: CardHeader;
  elements: CardElement[];
  expired?: boolean;       // 卡片是否已过期（操作过后置为 true）
}

interface CardHeader {
  title: string;
  subtitle?: string;
  /** 主题色：blue | green | yellow | red | grey */
  template?: 'blue' | 'green' | 'yellow' | 'red' | 'grey';
  icon?: string;           // 图标 URL 或内置图标名
}

/** 卡片元素联合类型 */
type CardElement =
  | DivElement
  | HrElement
  | ImageElement
  | ActionElement
  | FormElement
  | NoteElement;

/** 文本块 */
interface DivElement {
  tag: 'div';
  text: string;            // 支持 Markdown
  style?: 'primary' | 'secondary' | 'danger';
}

/** 分割线 */
interface HrElement {
  tag: 'hr';
}

/** 图片块 */
interface ImageElement {
  tag: 'image';
  src: string;
  alt?: string;
  width?: number | 'full';
}

/** 操作区（包含按钮等） */
interface ActionElement {
  tag: 'action';
  layout?: 'horizontal' | 'vertical';
  actions: (ButtonAction | SelectAction)[];
}

/** 按钮 */
interface ButtonAction {
  tag: 'button';
  key: string;             // 回传给服务端的标识
  text: string;
  type?: 'primary' | 'default' | 'danger';
  disabled?: boolean;
  confirm?: {              // 二次确认
    title: string;
    content: string;
  };
}

/** 下拉选择 */
interface SelectAction {
  tag: 'select';
  key: string;
  placeholder?: string;
  options: Array<{ label: string; value: string }>;
  multiple?: boolean;
}

/** 表单块（包含多个输入控件 + 提交按钮） */
interface FormElement {
  tag: 'form';
  submitKey: string;       // 提交时的 action key
  submitText?: string;
  fields: FormField[];
}

type FormField = InputField | SelectField | DateField | RadioField;

interface InputField {
  tag: 'input';
  key: string;
  label: string;
  placeholder?: string;
  required?: boolean;
  type?: 'text' | 'number' | 'email';
}

interface SelectField {
  tag: 'select';
  key: string;
  label: string;
  options: Array<{ label: string; value: string }>;
  multiple?: boolean;
  required?: boolean;
}

interface DateField {
  tag: 'date';
  key: string;
  label: string;
  format?: string;
  required?: boolean;
}

interface RadioField {
  tag: 'radio';
  key: string;
  label: string;
  options: Array<{ label: string; value: string }>;
  required?: boolean;
}

/** 备注/提示 */
interface NoteElement {
  tag: 'note';
  text: string;
  type?: 'info' | 'warning' | 'success' | 'error';
}
```

### 2.2 卡片示例

```json
{
  "type": "interactive_card",
  "cardId": "card_20240101_001",
  "header": {
    "title": "请确认您的操作",
    "template": "blue"
  },
  "elements": [
    {
      "tag": "div",
      "text": "您即将执行 **删除用户数据** 操作，此操作不可撤销。"
    },
    { "tag": "hr" },
    {
      "tag": "action",
      "layout": "horizontal",
      "actions": [
        {
          "tag": "button",
          "key": "confirm_delete",
          "text": "确认删除",
          "type": "danger",
          "confirm": {
            "title": "最终确认",
            "content": "确定要删除吗？"
          }
        },
        {
          "tag": "button",
          "key": "cancel",
          "text": "取消",
          "type": "default"
        }
      ]
    }
  ]
}
```

### 2.3 卡片渲染组件
```
CardRenderer
├── CardHeader        → 标题区（带色彩主题）
├── DivRenderer       → 文本（支持 Markdown）
├── HrRenderer        → 分割线
├── ImageRenderer     → 图片展示
├── ActionRenderer    → 按钮/选择器组
│   ├── ButtonAction  → 按钮（含二次确认）
│   └── SelectAction  → 下拉选择
├── FormRenderer      → 表单
│   ├── InputField    → 文本/数字输入
│   ├── SelectField   → 下拉选择
│   ├── DateField     → 日期选择器
│   └── RadioField    → 单选按钮组
└── NoteRenderer      → 备注提示
```

### 2.4 卡片交互回传
用户操作卡片后，通过 WebSocket 发送：
```json
{
  "type": "card_action",
  "sessionId": "session_xxx",
  "requestId": "req_xxx",
  "cardId": "card_20240101_001",
  "action": {
    "tag": "button",
    "key": "confirm_delete",
    "value": null
  }
}
```
表单提交时，value 为表单所有字段值的键值对：
```json
{
  "action": {
    "tag": "form",
    "key": "submit_form",
    "value": {
      "name": "张三",
      "department": "研发部",
      "date": "2024-01-01"
    }
  }
}
```

**操作完成后**：
- 卡片变为 `expired: true` 状态，显示操作结果摘要
- 禁用所有交互元素
- 展示"已提交"标记

## 3. 图表模块设计

### 3.1 图表 JSON 协议

```typescript
// src/types/chart.ts

interface ChartMessage {
  type: 'chart';
  messageId: string;
  /** 图表类型 */
  chartType: ChartType;
  /** 图表数据与配置 */
  payload: ImagePayload | TablePayload | G2ChartPayload | IframePayload;
}

type ChartType = 'image' | 'table' | 'line' | 'bar' | 'pie' | 'area' | 'scatter' | 'iframe';

/** 图片类型 */
interface ImagePayload {
  url: string;
  alt?: string;
  width?: number;
  height?: number;
  downloadable?: boolean;
}

/** S2 表格类型 */
interface TablePayload {
  /** 列定义 */
  columns: Array<{
    key: string;
    title: string;
    dataType?: 'text' | 'number' | 'date' | 'percent';
    width?: number;
    sortable?: boolean;
  }>;
  /** 数据行 */
  rows: Array<Record<string, any>>;
  /** 表格配置 */
  options?: {
    pagination?: boolean;
    pageSize?: number;
    exportable?: boolean;   // 支持导出 CSV
    totals?: boolean;       // 显示合计行
  };
}

/** G2 通用图表类型（折线/柱状/饼图/面积/散点） */
interface G2ChartPayload {
  /** G2 Spec 格式，直接传入 AntV G2 的 spec 配置 */
  spec: Record<string, any>;
  /** 图表高度，默认 300 */
  height?: number;
  /** 是否支持导出图片 */
  exportable?: boolean;
}

/** iframe 嵌入（方案 B） */
interface IframePayload {
  url: string;
  height?: number;
  allowFullscreen?: boolean;
}
```

### 3.2 G2 图表示例（折线图）

```json
{
  "type": "chart",
  "chartType": "line",
  "payload": {
    "spec": {
      "type": "line",
      "data": {
        "value": [
          { "date": "2024-01", "value": 120, "category": "销售额" },
          { "date": "2024-02", "value": 180, "category": "销售额" },
          { "date": "2024-03", "value": 150, "category": "销售额" }
        ]
      },
      "encode": {
        "x": "date",
        "y": "value",
        "color": "category"
      },
      "title": { "text": "月度销售趋势" }
    },
    "height": 320,
    "exportable": true
  }
}
```

### 3.3 表格示例

```json
{
  "type": "chart",
  "chartType": "table",
  "payload": {
    "columns": [
      { "key": "name", "title": "姓名" },
      { "key": "department", "title": "部门" },
      { "key": "score", "title": "评分", "dataType": "number", "sortable": true }
    ],
    "rows": [
      { "name": "张三", "department": "研发部", "score": 95 },
      { "name": "李四", "department": "产品部", "score": 88 }
    ],
    "options": {
      "pagination": true,
      "pageSize": 10,
      "exportable": true,
      "totals": false
    }
  }
}
```

### 3.4 图表组件架构

```
ChartRenderer（根据 chartType 分发）
├── ImageChart         → 图片（Ant Design Image，支持预览）
├── TableChart         → Ant Design Table（自适应高度，超 420px 内部滚动，支持导出 CSV）
├── G2Chart            → G2 统计图（@antv/g2，懒加载，高度上限 480px）
│   ├── line           → 折线图
│   ├── bar            → 柱状图
│   ├── pie            → 饼图
│   ├── area           → 面积图
│   └── scatter        → 散点图
└── IframeChart        → iframe 嵌入（含加载状态、全屏按钮，高度上限 560px）
```

> **注意**：TableChart 使用 Ant Design Table 实现（而非 @antv/s2-react），因 S2 v2 API 变更较大，以保证稳定性。

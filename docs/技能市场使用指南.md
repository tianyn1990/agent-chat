# 技能市场使用指南

## 功能概述

技能市场是 OpenClaw Web 客户端的核心功能之一，允许用户浏览、安装和管理各种 AI 技能，扩展 OpenClaw 的能力。

## 访问技能市场

1. 登录后，点击左侧边栏的 **Skills** 图标
2. 进入技能市场主页面

## 主要功能

### 1. 浏览技能

技能市场以卡片式布局展示所有可用技能，每个技能卡片包含：
- 技能图标
- 技能名称
- 简短描述
- 分类标签
- 版本号
- 安装状态

### 2. 搜索技能

在页面顶部的搜索框中输入关键词，可以实时搜索：
- 技能名称
- 技能描述
- 技能标签

**示例**：
- 搜索 "数据" 可以找到 "数据分析师" 技能
- 搜索 "SQL" 可以找到 "SQL 助手" 技能

### 3. 分类筛选

点击分类标签可以按类别筛选技能：
- **全部** - 显示所有技能
- **数据分析** - 数据处理、统计分析相关
- **写作助手** - 文档写作、内容创作相关
- **代码工具** - 编程、代码审查相关
- **办公自动化** - PPT、邮件等办公场景
- **沟通协作** - 沟通、协作相关
- **其他** - 其他类型技能

### 4. 查看技能详情

点击任意技能卡片，右侧会弹出详情抽屉，展示：
- 技能完整描述
- 主要功能特性列表
- 版本信息
- 作者信息
- 标签
- 主页链接（如有）

### 5. 安装技能

**未安装的技能**：
1. 在技能卡片上点击 **安装** 按钮，或
2. 打开技能详情，点击底部的 **安装** 按钮
3. 等待安装完成（显示 Loading 状态）
4. 安装成功后，技能卡片会显示 **已安装** 标识

**已安装的技能**：
- 技能卡片右上角显示绿色的 **已安装✓** 标识
- 按钮文本变为 **已安装**
- 在 "已安装技能" 区域单独展示

### 6. 卸载技能

1. 点击已安装的技能卡片，打开详情抽屉
2. 点击底部的 **卸载** 按钮
3. 等待卸载完成
4. 卸载成功后，技能恢复为未安装状态

### 7. 在对话中使用技能

对于已安装的技能，可以快速在对话中使用：
1. 打开技能详情抽屉
2. 点击 **在对话中使用** 按钮
3. 自动跳转到对话页面，并预填技能名称

## 常见问题

### Q: 如何知道哪些技能已安装？
A: 已安装的技能会在卡片右上角显示绿色的 **已安装✓** 标识，并且在页面顶部的 "已安装技能" 区域单独展示。

### Q: 安装技能需要多长时间？
A: 当前使用 Mock 数据，安装操作会模拟 500ms 的延迟。真实环境下，安装时间取决于网络状况和技能大小。

### Q: 可以同时安装多个技能吗？
A: 可以，但建议逐个安装，以便观察每个技能的安装状态。

### Q: 卸载技能会丢失数据吗？
A: 当前版本卸载技能不会影响已有的对话数据。技能配置数据的保留策略将在后续版本中完善。

### Q: 如何更新技能？
A: 技能更新功能将在后续版本中提供。

## 技术说明

### Mock 数据
当前版本使用 Mock 数据模拟技能市场功能，所有操作（安装、卸载）仅在前端状态中生效，不会持久化到后端。

### 真实接口对接
在阶段七（Mock 转真实接口）中，将对接以下 API：
- `GET /api/skills` - 获取技能列表
- `GET /api/skills/:id` - 获取技能详情
- `POST /api/skills/:id/install` - 安装技能
- `DELETE /api/skills/:id` - 卸载技能
- `PATCH /api/skills/:id/config` - 更新技能配置

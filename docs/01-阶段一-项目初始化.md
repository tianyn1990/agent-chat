# 阶段一：项目初始化与基础架构

## 1. 目标
- 搭建 Vite + React + TypeScript 项目脚手架
- 配置开发工具链（ESLint、Prettier、Husky）
- 集成 Ant Design、Zustand、React Router
- 建立基础目录结构和通用组件
- 配置 Vitest 测试环境

## 2. 详细任务

### 2.1 项目脚手架
```bash
# 使用 Vite 创建项目
npm create vite@latest . -- --template react-ts
```

### 2.3 Vite 配置
```typescript
// vite.config.ts
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import path from 'path';

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src'),
    },
  },
  css: {
    preprocessorOptions: {
      less: {
        javascriptEnabled: true,
      },
    },
  },
  server: {
    port: 3000,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true,
      },
      '/ws': {
        target: 'ws://localhost:8080',
        ws: true,
      },
    },
  },
});
```

### 2.4 路由配置
```typescript
// src/router/index.tsx
const routes = [
  {
    path: '/login',
    element: <LoginPage />,
  },
  {
    path: '/',
    element: <AuthGuard><MainLayout /></AuthGuard>,
    children: [
      { path: 'chat/:sessionId?', element: <ChatPage /> },
      { path: 'skills', element: <SkillsPage /> },
      { path: 'visualize/:sessionId', element: <VisualizePage /> },
      { index: true, element: <Navigate to="/chat" /> },
    ],
  },
];
```

### 2.5 基础布局组件
```
MainLayout
├── Sidebar (左侧边栏, 宽度 260px)
│   ├── Logo + 应用名称
│   ├── 菜单导航 (对话 / Skills)
│   ├── 会话列表 (对话菜单下)
│   └── 用户信息 (底部)
└── Content (右侧主内容区)
    └── <Outlet /> (路由子页面)
```

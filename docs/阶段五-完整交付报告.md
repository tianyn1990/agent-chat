## 🐛 Bug 修复

### Bug #1: 侧边栏布局问题
**问题描述**：切换到 Skills 页面时，用户信息从底部跑到了顶部

**根本原因**：
- `Sidebar.tsx` 中，当 `showExtra` 为 `false` 时，`.extra` 区域不渲染
- `.extra` 有 `flex: 1` 属性用于占据剩余空间
- 不渲染时，`.footer` 失去了推到底部的支撑

**解决方案**：
```tsx
// 修改前
{showExtra && extra && (
  <div className={styles.extra}>{extra}</div>
)}

// 修改后
<div className={styles.extra}>
  {showExtra && extra}
</div>
```

**验证结果**：✅ 已修复，用户信息始终在底部

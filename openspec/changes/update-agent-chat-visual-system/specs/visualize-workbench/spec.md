## ADDED Requirements

### Requirement: 沉浸式工作台外层视觉必须与主产品统一

系统 MUST 在保持沉浸式 `iframe-first` 工作台结构的前提下，使工作台的外层背景、悬浮控件、状态反馈和页面切换语义与主产品视觉系统保持一致。

#### Scenario: 从聊天页进入工作台

- **Given** 用户当前位于聊天页
- **When** 用户打开某会话的沉浸式执行状态工作台
- **Then** 工作台外层必须延续主产品的色板、控件语义和过渡反馈
- **And** 用户必须能够感知这是同一产品中的沉浸式场景，而不是割裂的外接页面

#### Scenario: 工作台展示真实 iframe

- **Given** 工作台已经打开并展示真实 `Star-Office-UI` iframe
- **When** 用户观察工作台外层壳层
- **Then** 悬浮控件必须保持极简
- **And** 外层非必要说明信息不得再次大面积占据真实办公室画面
